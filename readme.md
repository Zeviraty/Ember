# Ember

This is a remake of pokemon red in C

> [!WARNING]
> This project is unfinished and in alpha, the basic features aren't implemented yet

# Building

## Linux

Make sure you have sdl2 & sdl2_image installed.
then run `make`, the executable is `build/ember`.

## Windows

Make sure you have mingw installed, and then download and extract the sdl2 and sdl2_image libraries
Into the libs folder with these names:
 - `SDL2`
 - `SDL2_image`

then run `make win` the executable is `build/ember.exe`
